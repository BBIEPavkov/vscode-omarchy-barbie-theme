# Omarchy Barbie - VS Code Theme

A Barbie-inspired dark VS Code theme with deep violet-pink backgrounds, hot pink accents, sky blue class names, royal blue modifiers, and pastel yellow method names.

## Colors

| Element | Color |
|---------|-------|
| Background | `#2D0A22` (deep violet-pink) |
| Foreground | `#FF8EC8` (bold pink) |
| Keywords / Control flow | `#FF69B4` (pink) |
| Modifiers (`public`, `private`, `async`) | `#6A8FD8` (royal blue) |
| Class / Type names | `#87CEEB` (sky blue) |
| Method names | `#FFF0A0` (pastel yellow) |
| Strings | `#FF85C1` (light pink) |
| Numbers | `#FFF0A0` (pastel yellow) |
| Parameters | `#F48FB1` (soft pink, italic) |
| Comments | faded pink, italic |

## Install

Search for **Omarchy Barbie** in the VS Code Extensions Marketplace, or install manually:

1. Download this repo
2. Copy the folder into `~/.vscode/extensions/`
3. Reload VS Code and select **Omarchy Barbie** from the theme picker (`Ctrl + Shift + P` → Preferences: Color Theme)

## Pairs well with

The [Omarchy Barbie desktop theme](https://github.com/BBIEPavkov/omarchy-barbie-theme) for Arch Linux.
